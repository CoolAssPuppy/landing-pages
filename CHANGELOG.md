# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-02-14

### Added

- Fifteen landing page templates: lead generation, thank you, legal/rules, waitlist, resource hub, partner co-marketing, comparison, pricing, webinar, case study, product launch, demo request, job listing, event recap, and integration
- Data-driven page configuration system using TypeScript objects
- Standard shadcn/ui theming with CSS variables and dark/light mode
- HubSpot Forms/Contacts API integration
- Customer.io Track API integration with configurable events
- Secure form handling with CSRF protection, rate limiting, and input sanitization
- UTM parameter tracking with automatic capture and persistence
- Webinar platform URL detection for Zoom, GoToWebinar, Teams, Webex, Google Meet, YouTube Live, Livestorm, Demio, Crowdcast, and StreamYard
- Content-Security-Policy, HSTS, and other security headers
- Example pages for every template type
- Comprehensive documentation for customization and integration
